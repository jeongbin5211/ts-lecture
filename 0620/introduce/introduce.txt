타입스크립트 
- 2010년대 초에 마이크로소프트 내부에서 개발
- 2012년에 출시 및 오픈 소스화
- 자바스크립트의 상위 집합 or 타입이 있는 자바스크립트에서는

타입스크립트를 설명하는 네 가지 키워드
- 프로그래밍 언어 : 자바스크립트의 모든 구문과 타입을 정의하고 사용하기 위한 새로운 타입스크립트 고유 구문이 포함된 언어
- 타입 검사기 : 자바스크립트 및 타입스크립트로 작성된 일련의 파일에서 생성된 모든 구성 요소(변수, 함수 등)를 이해하고, 잘못 구성된 부분을 알려주는 프로그램
- 컴파일러 : 타입 검사기를 실행하고 문제를 보고한 후 이에 대응되는 자바스크립트 코드를 생성하는 프로그램
- 언어 서비스 : 타입 검사기를 사용해 비주얼 스튜디오 코드(VS Code)와 같은 편집기에 개발자에게 유용한 유틸리티 제공법을 알려주는 프로그램

VSCode 확장기능 추천 
// Prettier와 ESLint는 되도록 설치해주세요 //
// Prettier는 코드의 포맷팅을 담당하고, ESLint는 코드의 품질과 잠재적인 문제점을 확인하는 데 사용 //

1. Better Comments
-> 주석 꾸미기

2. Color Highlight
-> CSS적용 시 소스코드에서 색상 확인 가능

3. eslint
-> JavaScript 및 JSX 코드에 대해 정적 분석을 수행 / 일관성 있는 코드 작성, 잠재적 에러와 안티 패턴 찾기에 도움

4. indent-rainbow
-> 띄어쓰기/탭을 색상으로 구분

5. material Icon Theme & vscode-icons
-> 폴더 또는 파일의 아이콘 꾸미기 ( 둘 중 하나 택 )

6. Prettier
-> 코드 포매터, 코드를 일관된 스타일로 자동으로 포맷팅하는 도구
